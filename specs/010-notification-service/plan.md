# Implementation Plan: Phase 5.3 — Event-Driven Notification Service

**Branch**: `010-notification-service` | **Date**: 2026-03-09 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/010-notification-service/spec.md`

---

## Summary

Deploy a standalone Python FastAPI microservice (`todo-notification-service`) that subscribes to the `reminders` Kafka topic via Dapr declarative subscription. The main backend publishes reminder events when a task title contains a trigger keyword ("remind me" or "alert"). The notification service receives Dapr-pushed CloudEvents and logs a structured reminder message. No database, no auth, no UI — stateless consumer only.

---

## Technical Context

**Language/Version**: Python 3.13 (notification service) + Python 3.13 (backend extension)
**Primary Dependencies**: FastAPI 0.115+, uvicorn[standard] 0.30+ (notification service); httpx 0.27+ (backend events.py — already present)
**Storage**: None — notification service is stateless. No DB table changes.
**Testing**: Manual smoke test via kubectl logs — no automated tests in Phase 5.3
**Target Platform**: Minikube (Kubernetes 1.28+), Dapr 1.17+
**Performance Goals**: Reminder delivered within 5 seconds of task creation (SC-001)
**Constraints**: Never crash on malformed events (FR-006); fire-and-forget publish (FR-002 aligns with Phase 5.2 pattern)
**Scale/Scope**: 1 replica notification service pod; single reminder keyword check per task create

### Notification Service (`services/notification/app/main.py`)

```python
"""Notification Service — Phase 5.3 — [Task]: T-5.3.x"""
import logging
from fastapi import FastAPI, Request

logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(message)s")
logger = logging.getLogger(__name__)

app = FastAPI(title="todo-notification-service", version="1.0.0")

_VALID_ACTION = "reminder"


@app.post("/on-reminder", status_code=200)
async def on_reminder(request: Request) -> dict:
    """Dapr-invoked endpoint — processes reminder events from the reminders topic.

    Returns HTTP 200 always to acknowledge Dapr delivery (prevent retry storm).
    Malformed events are logged as WARNING and skipped.
    """
    try:
        body = await request.json()
        payload = body.get("data", body)  # unwrap CloudEvents envelope (R-003)

        task_title = payload.get("task_title", "").strip()
        user_id = payload.get("user_id", "").strip()

        if not task_title or not user_id:
            logger.warning(
                "on_reminder: missing task_title=%r or user_id=%r — skipping",
                task_title, user_id,
            )
            return {}

        logger.info(
            "[REMINDER]: Hey User %s, your task \"%s\" is due now!",
            user_id, task_title,
        )

        # ── Placeholder: future Email / Push notification ──────────────────
        # await send_email_notification(user_id, task_title)
        # await send_push_notification(user_id, task_title)
        # ──────────────────────────────────────────────────────────────────

    except Exception as exc:
        logger.error("on_reminder: unhandled error: %s", exc)
        # Still return 200 — prevent Dapr infinite retry loop

    return {}


@app.get("/healthz")
def health() -> dict:
    return {"status": "ok"}
```

### events.py Extension (`backend/app/logic/events.py`)

Add below existing `publish_task_event`:

```python
# [Task]: T-5.3.x — publish_reminder_event
_DAPR_REMINDER_URL = "http://localhost:3500/v1.0/publish/todoai-pubsub/reminders"

def publish_reminder_event(task_id: str, task_title: str, user_id: str) -> None:
    """Publish a reminder event to the reminders topic via Dapr sidecar.

    Fire-and-forget: never raises. 3 retries, exponential backoff (Principle XV).
    Safe when Dapr absent (HF Space) — ConnectionRefused caught silently.
    """
    if not task_id or not task_title or not user_id:
        logger.warning("publish_reminder_event: missing required field — skipping")
        return

    payload = {
        "action": "reminder",
        "task_id": task_id,
        "task_title": task_title,
        "user_id": user_id,
        "timestamp": datetime.now(timezone.utc).isoformat(),
    }
    backoff = _INITIAL_BACKOFF_S
    for attempt in range(1, _MAX_RETRIES + 1):
        try:
            response = httpx.post(_DAPR_REMINDER_URL, json=payload, timeout=5.0)
            if response.status_code == 204:
                logger.debug("publish_reminder_event: published task_id=%s", task_id)
                return
            logger.error("publish_reminder_event: status=%d attempt=%d", response.status_code, attempt)
        except Exception as exc:
            logger.error("publish_reminder_event: attempt=%d failed: %s", attempt, exc)
        if attempt < _MAX_RETRIES:
            time.sleep(backoff)
            backoff = min(backoff * 2, _MAX_BACKOFF_S)
    logger.error("publish_reminder_event: EXHAUSTED retries task_id=%s", task_id)
```
