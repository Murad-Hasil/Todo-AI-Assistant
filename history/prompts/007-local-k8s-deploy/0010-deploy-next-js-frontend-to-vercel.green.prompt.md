## Prompt

Deploy the Next.js Frontend to Vercel using the Vercel CLI (npx).

MISSION: Launch the Todo AI Frontend and ensure it is ready to communicate with a future production backend.

STRICT DEPLOYMENT RULES:
1. **PROJECT LINKING:**
   - Navigate to `/todo-web-app/frontend`.
   - Run `npx vercel link --yes --token <YOUR_VERCEL_TOKEN>`.
2. **MONOREPO CONFIGURATION:**
   - Set the Project Name: "todo-ai-frontend".
   - Framework: Next.js.
   - Root Directory: `todo-web-app/frontend`.
3. **HANDLING DEPENDENCY CONFLICTS:**
   - Set an environment variable in Vercel: `NPM_CONFIG_LEGACY_PEER_DEPS=true`.
4. **ENVIRONMENT VARIABLES:**
   - `BETTER_AUTH_SECRET`: 32-char key from .env.local.
   - `BETTER_AUTH_URL`: production URL generated by Vercel.
   - `NEXT_PUBLIC_API_URL`: "https://api-placeholder.com".
5. **PRODUCTION BUILD:**
   - Execute `npx vercel deploy --prod --token <TOKEN> --yes`.
